## Quick Start

Install node and npm on your machine, after that clone repo with:
```bash
git clone https://github.com/arturgoms/Dashboard/
```
 Enter in directory
```bash
cd Dashboard
```
Install depedencies
```bash
npm install
```
Run it:
```bash
npm start
```
